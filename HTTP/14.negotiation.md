# 협상(콘텐츠 네고시에이션)
- 클라이언트가 선호하는 표현 요청
-  Accept : 클라이언트가 선호하는 미디어 타입 전달
- Aceept-Charset: 클라이언트가 선호하는 문자 인코딩
- Accept-Encoding: 클라이언트가 선호하는 압축 인코딩
- Accept-Language: 클라이언트가 선호하는 자연 언어
- 협상 헤더는 요청시에만 사용

## 1.1 협상과 우선순위1
- 0~1, 클수록 높은 우선순위
- Quality Values(q) 값 사용
- 생략하면 1

## 1.2. 협상과 우선순위2
- 구체적인 것이 우선한다.

## 1.3. 협상가 우선순위3
- 구체적인 것을 기준으로 미디어 타입을 맞춘다.
